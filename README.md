- Responsive design with Bootstrap 5
- Multilingual support (ID/EN)
- Fixed header navigation
- Image carousel
- Contact form with validation
- Registration system
- Component-based architecture
- Clean and optimized code structure

## Setup & Installation

1. Clone the repository:
```bash
git clone [repository-url]
```

2. Set up a local server (e.g., XAMPP, WAMP)

3. Configure backend paths in:
   - process_register.php
   - app.js component loader

4. Update asset paths if needed

## Development Guidelines

1. **CSS Organization**
   - Use CSS variables for consistent theming
   - Follow BEM naming convention
   - Maintain responsive design principles

2. **JavaScript Best Practices**
   - Use ES6+ features
   - Implement error handling
   - Follow module pattern
   - Maintain clean code structure

3. **Component Development**
   - Keep components independent
   - Use data attributes for translations
   - Maintain consistent styling
